# camila-ahur LP

Landing page da Camila Àhur (Psicanálise Junguiana).
Projeto standalone com git próprio.

## Stack
- React 19 + Vite 6 + Tailwind v4 + Framer Motion v12
- Fontes: Lora (display) + Nunito Sans (sans)

## Comandos
- `npm run dev` — dev server
- `npm run build` — build para produção
- `git push origin main` — deploy automático via Vercel
